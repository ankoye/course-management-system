springboot+vue项目实战
--------------------
#### 项目简介
本项目是模仿[云班课](https://www.mosoteach.cn/)实现的一个课程管理系统。

#### 使用技术
前端：vue + element-ui <br>
后端：springboot + shiro + mybatis + druid + swagger
